Apply manifest

## 用法

```console
Usage:
  nhctl apply [NAME] [MANIFEST] [flags]
```

## 标志

```console
Flags:
  -h, --help   help for apply

Global Flags:
      --auth-check          pre check the nocalhost commands permissions, return yes represent having enough permissions to call the command
      --debug               enable debug level log
      --kubeconfig string   the path of the kubeconfig file
  -n, --namespace string    kubernetes namespace
```
